# 排行榜工具資料夾 (Ranking Tool Workspace)

本目錄為專用於 **「社交博弈遊戲排行榜配置」** 的工作空間，整合了原始資料基準與視覺化生成工具。

## 📁 目錄概覽
此資料夾包含營運端用於管理、編輯及生成排行榜 JSON 設定檔的所有必要資源。

## 🎯 核心功能
1. **設定生成**：透過 `RankConfigGenerator.html` 產生符合伺服器規範的 JSON 檔案。
2. **資料校對**：參考 `260213_round_infos.py` 中的正式資料格式，確保欄位對應正確。
3. **智慧修復**：工具支援自動修正獎勵 JSON 的格式錯誤（如多餘逗號、型態不符）。

## 📄 重點檔案說明
- **`RankConfigGenerator.html`**: 
  - 核心工具檔案，支援 24 小時制時間設定。
  - 具備「多檔活動清單」管理功能，可一次編輯整週活動。
  - 支援「整批匯入」與「智慧獎勵修復」。
- **`260213_round_infos.py`**: 
  - 伺服器端提供的正式 Python 資料來源，包含了 `round_id`、`prizes`、`bq_extra_params` 等關鍵欄位的範例。

## 💡 操作建議
- **開始工作**：直接開啟 `RankConfigGenerator.html` 並點擊「📥 匯入正式資料」，將 `.py` 檔案內的 JSON 內容貼上進行編輯。
- **時間設定**：本工具已強制使用 24 小時制，選取時間後會自動轉換為系統要求的 `YYYYMMDDHHMM` 格式。
- **獎勵編輯**：若有大量名次獎勵，建議使用「JSON 貼上」模式並利用「智慧解析」功能進行自動格式校正。
